URL 설계

URL (Method) 설명 반환 데이터
/replies (POST) 특정한 게시물의 댓글 추가 {'rno':11} - 생성된 댓글의 번호
/replies/list/:bno (GET) 특정 게시물(bno)의 댓글 목록
'?' 페이지 번호를 추가해서 댓글 페이징 처리		 PageResponseDTO를 JSON으로 처리
/replies/:rno (PUT) 특정한 번호의 댓글 수정 {'rno':11} - 수정된 댓글 번호
/replies/:rno (DELETE) 특정한 번호의 댓글 삭제 {'rno':11} - 삭제된 댓글 번호
/replies/:rno (GET) 특정한 번호의 댓글 조회 댓글 객체를 JSON으로 변환한 문자열
